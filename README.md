# weather-app

A modern weather forecasting application built with Next.js that provides real-time weather data and forecasts.

## Features

- Real-time weather conditions for any location
- 7-day weather forecast
- Responsive design for all devices
- Temperature unit switching (Celsius/Fahrenheit)
- Location search functionality
- Visual weather representations

## Tech Stack

- Framework: Next.js 14 (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- UI Library: React
- API: Open-Meteo (free weather API)

## API Integration

The application uses the Open-Meteo API, which provides:

- Free weather data without API keys
- Global weather coverage
- 7-day forecast data
- Multiple weather parameters (temperature, precipitation, wind, humidity)
- Hourly and daily forecasts

## Getting Started

### Prerequisites

- Node.js 18 or later
- npm, yarn, or pnpm

### Installation

1. Clone the repository:

```
git clone https://github.com/alinidi/weather-app.git
cd weather-app
```

2. Install dependencies:

```
npm install
```

3. Run the development server:

```
npm run dev
```

4. Open http://localhost:3000 in your browser.
